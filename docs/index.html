Lakuna Discord Bot

Lakuna is a Discord bot with a variety of useful commands, focused on being lightweight, aesthetically pleasing, and non-intrusive.

Commands
Name Description
about Displays bot information.
help Gives the user information about bot commands.
invite Generates an invite link to add the bot to other guilds.
lol Displays information about a League of Legends account.
rr Creates a message which allows users to toggle specified roles.
view Displays information about a Discord object.
